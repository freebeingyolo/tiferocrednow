# 万达康运动健康

## 模块

```
app				#入口
lib_base		#基础模块
lib_ble			#蓝牙
lib_res			#公共资源
lib_step		#计步
```





## 第一期功能点

| 扫码       | 扫描设备二维码进行跳转至绑定设备页面                         | 基于zxing二维码扫描库二次开发，扫描获取设备信息（设备名、设备类型、mac地址），扫描后跳转绑定页面 | zxing/zbar/scankit                                           |      |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | ---- |
| 绑定设备   | 展示设备配网步骤，配合设备操作进行配网设置                   | 通过扫码的得到的设备mac地址开启ble连接                       |                                                              |      |
| 体重信息   | 展示当前体重、目标体重、相差数量、体型，可对当前体重和目标体重进行设置；     当连接过体脂秤设备后，以体脂秤最近测量的体重作为当前体重 | 蓝牙BLE连接后，通过开启体重信息的订阅消息，监听设备推送，获取推送消息后展示 | 权限检测和提示     蓝牙：     https://blog.csdn.net/nanjoh/article/details/93062114     com.vise.xiaoyaoyou:baseble:2.0.6     cn.wandersnail.ble |      |
| 今日步数   | 展示手机通知的步数、公里数、消耗卡路里数，目标步数的进度，可对目标步数进行设置 | 应用后台常驻一个服务统计用户步数；                           | 计步资料：     https://blog.csdn.net/u013700040/article/details/67633634 |      |
| 设备信息   | 展示设备名称与展示设备连接状态，点击进入设备控制页           | 通过监听蓝牙BLE状态获取连接状态                              |                                                              |      |
| 体脂秤     | 当前体重**（公斤）、健康建议（文案待定）、您的体型：偏瘦/标准/肥胖、综合得分、肥胖度、体重、BMI、脂肪含量、骨骼肌率、骨骼肌重量、肌肉率、肌肉重量、内脏脂肪、水份、水含量、基础代谢、骨量、蛋白质、去脂体重、身体年龄 | 通过蓝牙BLE订阅获取体脂秤数据                                |                                                              |      |
| 健腹轮     | 当前连接状态、使用时长、开启/关闭计数、电量、运动次数、当前热量消耗、玩法推荐 | 通过蓝牙BLE订阅交互                                          |                                                              |      |
| 视频列表   | 展示10个教程视频缩略图与播放按键，点击可弹出视频播放弹窗     | 通过内置url，打开指定网页                                    | jetpack播放器                                                |      |
| 健身好店铺 | 展示2个店铺图文信息，点击可跳转到APP店铺主页                 | 通过内置url，打开指定网页                                    |                                                              |      |
| 运动好产品 | 展示6个产品图文信息，点击可跳转到APP商品详情页               | 通过内置url，打开指定网页                                    |                                                              |      |
| 我的设备   | 查看已添加的设备并对设备进行设备名称修改，设备型号、MAC地址查看，固件更新，设备删除 | 通过蓝牙订阅消息，与设备交互                                 |                                                              |      |
| 关于APP    | 品牌展示                                                     | 通过预置品牌数据，展示品牌详情                               |                                                              |      |
